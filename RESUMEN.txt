2. INSTALACIÓN Y PUESTA EN MARCHA
================================================================================

Paso 1: Instalar dependencias
  cd Cucu/backend
  npm install

Paso 2: Iniciar servidor
  npm start
  
El servidor levanta en: http://localhost:3000

6. API ENDPOINTS (Backend)
================================================================================

Agendamientos:
  GET    /api/appointments              → Obtener todos
  POST   /api/appointments              → Crear nuevo
  GET    /api/appointments/day/:date    → Obtener por fecha
  DELETE /api/appointments/:id          → Eliminar agendamiento

Servicios:
  GET    /api/appointments/services     → Listar todos
  POST   /api/appointments/services     → Crear servicio
  DELETE /api/appointments/services/:id → Eliminar servicio

Configuración:
  GET    /api/appointments/config/settings    → Obtener configuración
  PUT    /api/appointments/config/settings    → Guardar configuración


7. BASE DE DATOS (SQLite3)
================================================================================

Tabla: appointments
  - Guarda todos los agendamientos de clientes
  - Campos: nombre, teléfono, email, servicio, fecha, hora, notas, estado

Tabla: services
  - Guarda los servicios disponibles
  - Campos: nombre, duración (minutos)
  - Sin precios (removidos)

Tabla: settings
  - Guarda la configuración del negocio
  - Horarios de cada día de la semana
  - Información del propietario

Tabla: available_times
  - Reserva de espacios de tiempo disponibles
  - Gestión de horarios libres


8. FLUJO DE USO
================================================================================

Para un CLIENTE:
  1. Entra a http://localhost:3000/
  2. Llena sus datos (nombre, teléfono, email)
  3. Selecciona un servicio (Ej: Corte de Cabello)
  4. Elige una fecha disponible
  5. Elige una hora disponible
  6. Revisa el resumen
  7. Hace clic en "Agendar Consulta"
  8. Recibe confirmación

Para un ADMINISTRADOR:
  1. Entra a http://localhost:3000/admin.html
  2. Redirige automáticamente a /login.html si no está logueado
  3. Ingresa usuario: admin y contraseña: cuba
  4. Accede al panel con 4 vistas:
     - Hoy: Ver agendamientos del día actual
     - Todos: Ver histórico completo
     - Buscar: Buscar agendamientos específicos
     - Configurar: Gestionar servicios, horarios e info del negocio
